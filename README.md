# Real eSwipe

## Описание

Real eSwipe е система за публикуване, търсене и резервиране на апартаменти. Проектът е разработен на Java и демонстрира използването на обектно-ориентирано програмиране, работа с роли на потребители, бизнес логика и автоматизирано тестване чрез Cucumber.

Системата поддържа различни типове потребители и позволява:

* Търсене на апартаменти по град, район и наличност.
* Резервиране на свободни апартаменти.
* Публикуване на нови обяви от потребители с роля Host.
* Управление на роли на потребители.
* Валидация на въведените данни.

## Основни функционалности

### Търсене на апартаменти

Потребителят може да търси апартаменти чрез:

* Град
* Район
* Статус (Свободен / Зает)

### Резервиране на апартамент

При резервиране системата проверява:

* Дали е въведена дата
* Дали датата е валидна
* Дали апартаментът е свободен
* Дали потребителят е успешно удостоверен

### Публикуване на обява

Само потребители с роля Host могат да публикуват обяви.

Системата валидира:

* Цена
* Град
* Наличие на снимка
* Роля на потребителя

---

## Роли

### User

Стандартен потребител, който може:

* Да търси апартаменти
* Да резервира апартаменти
* Да управлява профила си

### Host

Потребител, който може:

* Да публикува обяви
* Да управлява собствените си обяви

### Admin

Администратор с права за:

* Управление на потребители
* Одобряване на обяви

## Тестване

Проектът използва Cucumber за Behavior Driven Development (BDD).

Покрити са следните сценарии:

### Booking

* Успешно резервиране
* Резервиране на зает апартамент
* Липсваща дата
* Невалидна дата
* Несъществуващ потребител
* Грешна парола

### Listing

* Успешно публикуване
* Публикуване без снимка
* Публикуване без роля Host
* Невалидна цена
* Неподдържан град

### Search

* Търсене по град
* Търсене по район
* Търсене по статус
* Комбинирано търсене
